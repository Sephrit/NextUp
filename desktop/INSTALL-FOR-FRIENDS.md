# Installing Next Up

Thanks for testing Next Up! It's a private watch tracker — everything stays on your own computer.

## Windows

1. From the link I sent you, download the file ending in **`-setup.exe`**
   (something like `Next Up_0.2.0_x64-setup.exe`).
2. Double-click it.
3. **Windows may show a blue "Windows protected your PC" box.** That's normal — the
   app just isn't code-signed yet (signing costs money I haven't set up for a test build).
   Click **More info**, then **Run anyway**.
4. Follow the installer. If Windows asks to install the **Edge WebView2** component,
   let it — the app needs it and it installs automatically.
5. Launch **Next Up** from the Start menu.

That's it. On first run it'll ask who's watching and let you add a movie or show.

## macOS (if you're on a Mac instead)

1. Download the **`.dmg`** — the **Apple Silicon** one for M1/M2/M3/M4 Macs, or the
   **Intel** one for older Macs. If unsure, it's almost certainly Apple Silicon.
2. Open the `.dmg` and drag **Next Up** into **Applications**.
3. The first time, **right-click the app → Open** (don't just double-click), then click
   **Open** in the dialog. macOS blocks unsigned apps on a normal double-click; the
   right-click-Open is a one-time approval.

## Sending feedback

Anything that felt broken, confusing, or slow is useful. A screenshot helps. Your
library never leaves your machine unless you use **Settings → Export JSON backup**.
