# Changelog

## 0.2.1

- Simplified installers to one file per OS (NSIS on Windows, DMG on macOS).
- Added a getting-started guide with screenshots and add-a-title examples.
- Tagged releases now publish automatically (still marked prerelease).

## 0.2.0 - Public beta

- Added a Windows/macOS Tauri application with local migration and OS credential storage.
- Added solo-to-club profiles, sealed per-person ratings, reviews, favorites, and rewatch votes.
- Added Moviedex progress, timeline, genres, Watchable services, filters, and sorting.
- Added key-free TVmaze show search, bulk episode import, and full-row season expanders.
- Made Watchmode and the local AI/MCP package independent optional integrations.
- Added GitHub CI, installer workflows, security scanning, and tester documentation.
